# WpMVC Request Validator

A robust and extensible request validation system for WordPress plugins. It features a fluent, rule-based syntax inspired by Laravel, supporting complex validation scenarios, form requests, and seamless integration with `WP_REST_Request`.

For a full list of available features and detailed usage guides, please refer to the [Validation Documentation](https://wpmvc.com/docs/request-validator).
